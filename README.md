# Queues

## Feature Tasks

- Create 3 Standard Queues
    - QueueA
    - QueueB
    - QueueC
- Create command line applications in Java that use these queues
    - Queue Publisher
    - Sends a message to a queue, using its ARN or URL
    - Queue Client
    - Receives messages from a Queue (by ARN) and displays them
